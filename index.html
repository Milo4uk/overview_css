Java обзор

Java — строго типизированный объектно-ориентированный язык программирования общего назначения, разработанный компанией Sun Microsystems (в последующем приобретённой компанией Oracle). Приложения Java транслируются в байт-код, поэтому они могут работать на любой архитектуре с виртуальной Java-машиной. Дата официального выпуска — 23 мая 1995 года.

Java программирование

Java является объектно-ориентированным языком. Однако он поддерживает примитивные типы данных (int, char и т. д.). Коды Java компилируются в байт-код, который запускается на виртуальной машине Java (JVM), независимо от базовой архитектуры.
